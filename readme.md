# docker-texlive-pandoc

Basic image for texlive and pandoc 2.4.
Based on Ubuntu Xenial.

**Texlive packages:**

* texlive-xetex
* texlive-latex-base
* texlive-math-extra
* texlive-latex-extra
* texlive-lang-cyrillic

**LaTeX packages:**

* latex-xcolor

**apt packages:**

* fonts-cmu
* lmodern
* curl (for pandoc setup)

**Pandoc filters:**

* pandoc-crossref
